# Reusability Model

TaPTaP engine is designed for reuse through JSON configuration.

Developers do not need to modify engine code.

They only change:

- Entities
- Physics rules
- Game parameters
- Input mapping

Example:

Different games are possible:

1. Flappy Bird
2. Runner Game
3. Tap Dodger
4. Jump Platformer

All use the same engine.

Game behaviour is defined in JSON config.
